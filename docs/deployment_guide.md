# アプリを他のユーザーに共有・公開する方法

作成した「Go/No-Go Test」アプリを他の人に使ってもらうための主な方法を3つ紹介します。

## 1. 同じWi-Fi内の人に一時的に見せる（一番手軽）
PCと同じWi-Fiに繋がっているスマホやPCからアクセスできます。

1. ターミナルで以下のコマンドを実行します。
   ```bash
   npm run dev -- --host
   ```
2. 表示される `Network: http://192.168.x.x:5173` というURLを、他の人のスマホやPCのブラウザに入力してもらうと、あなたのPC上で動いているアプリを操作できます。
   * ※ あなたのPCが起動していて、コマンドが動いている間だけ使えます。

---

## 2. インターネット上に公開する（推奨）
ずっと使えるURLを発行する方法です。**Vercel** や **Netlify** という無料サービスを使うのが最も簡単です。

### 手順（Vercelの場合）
1. プロジェクトをビルド（本番用ファイル作成）します。
   ```bash
   npm run build
   ```
2. `dist` というフォルダが作成されます。これが「完成品」です。
3. [Vercel](https://vercel.com/) にアクセスし、アカウント作成（GitHubなどでログイン）。
4. ダッシュボードに「dist」フォルダをドラッグ＆ドロップするだけで公開できます。

---

## 3. ファイルとして渡す
インターネットを使わず、USBメモリなどで渡したい場合です。ただし、Reactアプリは単にHTMLをダブルクリックしても動きません。

もしファイルとして渡して動かしてもらうには、相手のPCにも Node.js 環境が必要です。
1. フォルダごと相手に渡す。
2. 相手のPCで `npm install` -> `npm run dev` をしてもらう。
（これは少しハードルが高いかもしれません）

---

### おすすめ
まずは **方法1（--host）** で近くの人に試してもらい、本格的に配布したい場合は **方法2（Vercelなど）** を試すのが良いでしょう。
